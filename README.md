This depository contains the code for paper "Hypernetwork-based policies for neural-ordinary-differential-equation control"

To learn the policy for linear system, run the jupyter file NNC/Train_linear.ipynb, for testing use: NNC/Test_linear.ipynb
To learn the policy for Kuramoto system, run the jupyter file NNC/Train_Kuramoto.ipynb, for testing use: NNC/Test_kuramoto.ipynb

To use LQR: OC/Linear/LQR.ipynb
To use AGM and v-AGM: OC/AGM/AGM.ipynb and OC/AGM/v-AGM.ipynb

To use MT-PPO: ppo/ppo.ipynb
